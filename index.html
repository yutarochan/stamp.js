How to Use

1. Add stamp.js

Append stamp.js at the end of your page for faster loading times.

<script src="stamp.js"></script>

2. Load Stampsheets

Stampsheets are collections of stamp images which encoded in base64 and saved in a JSON file. These files are then loaded into the API using the following method:

var stampsheets = [ "stamps/inukei_joshi.json" ];
var stampjs = new StampJS(stampsheets);

3. Start Stampin'!

Use the following directive and call the stamps through filling in the appropriate parameters like sheetID and name. It's that easy!

<stamp-js sheetID="" name=""></stamp-js>
